# Ultrasonic Radar System

## Overview
This project is an Arduino-based Ultrasonic Radar System that detects nearby objects using an HC-SR04 ultrasonic sensor. A servo motor rotates the sensor, and the detected objects are displayed on a radar screen using Processing IDE.

## Components Used
- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- Servo Motor (SG90)
- Jumper Wires
- Breadboard
- Processing IDE

## Software Used
- Arduino IDE
- Processing IDE

## Features
- 180° radar scanning
- Real-time object detection
- Distance measurement
- Radar visualization on computer
